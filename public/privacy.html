Privacy Policy

Effective date: April 6, 2026

1. What This App Does

Live Visitor Geo Pulse for Ecwid is a static, admin-only dashboard that displays aggregated order and product metrics to the authenticated store owner inside the Ecwid admin panel.

2. Data We Collect

This app does not collect, store, or transmit personal data to any external service.

  • No server-side storage: The app has no database, backend server, or persistent data store.
  • No visitor tracking: The app does not inject storefront scripts, pixels, cookies, or fingerprinting code.
  • No third-party analytics: The app does not use Google Analytics, Mixpanel, Segment, or any tracking SDK.

3. Data Accessed via the Ecwid API

When opened inside the Ecwid admin iframe, the app reads the following from the Ecwid REST API v3 using the token provided by the admin panel:

  • Store profile (store name, currency)
  • Recent orders (order number, total, country, fulfillment status, customer name)
  • Product catalog (name, SKU, stock quantity, enabled status)

This data is fetched in real time, rendered in the browser, and never sent to any server controlled by the app developer. It stays entirely within the store owner's browser session.

4. Browser-Local Preferences

Dashboard preferences (lookback window, low-stock threshold, auto-refresh interval) are stored in localStorage in the owner's browser, namespaced by store ID. These preferences never leave the browser.

5. Cookies

This app does not set any cookies.

6. Data Sharing

This app does not share any data with third parties.

7. Data Retention

No data is retained on any server. Browser-local preferences persist until the user clears their browser storage or uninstalls the app.

8. Your Rights

Since no personal data is collected or stored by this app, there is no data to access, correct, or delete. If you have questions, contact the support channel listed in the marketplace listing.

If this policy is updated, the effective date above will be revised. Continued use of the app after changes constitutes acceptance of the updated policy.

10. Contact
