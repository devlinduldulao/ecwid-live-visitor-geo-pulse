Getting Started

Live Visitor Geo Pulse is an admin-only dashboard. After installing the app from the Ecwid App Market, open it from your Ecwid control panel. The dashboard loads automatically with your store's live data — no setup or registration required.

Frequently Asked Questions

The dashboard shows "Preview" data instead of my real store data

If Preview Mode is active, click Disable Preview Data in the Dashboard Controls section. If you are viewing the app outside the Ecwid admin iframe, it will default to preview mode because the Ecwid auth payload is not available.

Metrics show "--" or "Loading"

This typically means the Ecwid API could not be reached. Verify your internet connection and that you are viewing the app inside the Ecwid admin panel. Click Refresh Now to retry.

How do I change the lookback window or low-stock threshold?

Scroll to Local Preferences in the dashboard, adjust the values, and click Save Preferences. Preferences are stored only in your current browser for this store.
