Architecture

No extra infrastructure

This Ecwid version is a static admin app. It reads live store data from the Ecwid API in the owner dashboard and does not run a Node.js service, Redis cache, or custom database.

Scope

For the business owner

This surface is intentionally admin-first. It does not inject a storefront widget or expose visitor-facing UI.
