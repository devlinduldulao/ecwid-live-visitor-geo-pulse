Getting Started

1
Install the app

Add Live Visitor Geo Pulse from the Ecwid App Market to your store.

2
Open from your Ecwid admin

Go to your Ecwid admin panel → Apps → Live Visitor Geo Pulse. The dashboard loads inside the admin iframe automatically.

3
View your real store data

Once open inside Ecwid admin, the dashboard fetches your real orders, products, and geographic data. No setup needed — it just works.

Architecture

No extra infrastructure

This Ecwid version is a static admin app. It reads live store data from the Ecwid API in the owner dashboard and does not run a Node.js service, Redis cache, or custom database.

Scope

For the business owner

This surface is intentionally admin-first. It does not inject a storefront widget or expose visitor-facing UI.
